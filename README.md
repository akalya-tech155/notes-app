# Notes App

## Description
A simple Notes App built using HTML, CSS, and JavaScript.

## Features
- Create notes
- Delete notes
- Search notes
- Save notes using Local Storage
- Dark mode

## Technologies Used
- HTML
- CSS
- JavaScript
- Git
- GitHub

## Branches Created
- feature/create-note
- feature/delete-note
- feature/search-notes
- feature/localstorage
- feature/dark-mode

## How to Run
1. Download the project
2. Open index.html in a browser
3. Start creating notes
